# Predictive Model of Cardiovascular Risk 🫀

Este repositório contém o código-fonte desenvolvido para o artigo científico submetido à **RECISATEC**, focado na predição de riscos cardiovasculares utilizando técnicas de Machine Learning.

## 📝 Descrição do Projeto
O objetivo deste projeto é analisar dados clínicos para identificar padrões que indiquem a probabilidade de desenvolvimento de doenças cardiovasculares. O modelo utiliza algoritmos de classificação para auxiliar na tomada de decisão médica preventiva.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Bibliotecas:** * Pandas (Manipulação de dados)
  * Scikit-learn (Criação do modelo preditivo)
  * Matplotlib/Seaborn (Visualização de dados)
* **Ambiente:** Google Colab / Jupyter Notebook
